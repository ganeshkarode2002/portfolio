Project 2

" MOVIE RECOMMENDATION SYSTEM "
● Developed a Web based Movie Recommendation System using ML .
● Machine learning model was implemented using KNN classifer for classification of the movies.
● For website part Streamlit framework was used.

Reference Link

Project Overview

The Movie Recommendation System project aimed to develop a web-based platform that could provide personalized movie recommendations to users. Machine learning techniques were employed to build a recommendation model using the K-Nearest Neighbors (KNN) classifier. The model was trained on a dataset containing movie features such as genre, director, actors, and user ratings. By analyzing the similarities between movies based on these features, the model could suggest movies that were likely to be of interest to a particular user.

To create the user interface and deploy the recommendation system, the project utilized the Streamlit framework. Streamlit is a popular Python library that allows for the creation of interactive web applications with minimal coding. It provided an intuitive and user-friendly interface for users to input their preferences, view recommended movies, and explore additional details about each movie. Overall, the Movie Recommendation System project successfully implemented a web-based platform that leveraged machine learning algorithms to deliver personalized movie recommendations. By utilizing the KNN classifier and the Streamlit framework, the system provided an engaging and efficient way for users to discover new movies based on their individual preferences.
